What's the word, hummingbird?

Create stunning word clouds right in your browser using nothing but HTML5. This is meant to be both a showcase and an experiment of basic HTML5 technologies. Yay!

The source code is available on github under the MIT License. Feel free to clone the repo and play around with the fonts, colors, and other parameters. Suggestions on how I can improve this are welcomed.

If you've managed to read this far, then you deserve to see the fullscreen mode (if your browser supports it).

Based on Wordle by Jonathan Feinberg. I highly recommend checking them out. Fonts are from Google Web Fonts.
